网络问题

在以上步骤中，涉及下载和上传的步骤都可能出现网络问题。这很玄学

可能的解决方法：

  1. 重新开启/关闭梯子
  2. 更换下载的镜像源：国内npm源镜像（npm加速下载） 指定npm镜像_npm 国内镜像-CSDN博客
  3. 等一等。有时候你只需要等一等（比如我吃了个午饭后，hexo d就正常上传了）
  4. 在stackoverflow上看到一个集大成的回答

最终我通过换源+”npm config rm proxy”成功弄通了npm install
